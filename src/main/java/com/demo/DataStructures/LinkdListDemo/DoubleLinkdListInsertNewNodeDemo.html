CPU 执行时间轴

Step 0: 初始状态

此时 curr.next 存储的是 0x200。

Step 1: 参数快照 (Right Side)

new Node(99, curr.next, curr)

CPU 读取 curr.next。拿到值：0x200。

Step 2: 堆内存实例化

新节点 0x999 诞生，它的内部 next 已经被填上了刚才拿到的 0x200。

Step 3: 赋值更新 (Left Side)

curr.next = 0x999;

直到此时，curr.next 的值才从 0x200 变为 0x999。

Step 4: 下一行代码启动

curr.next.next.prev = curr.next;

注意！现在去读 curr.next，拿到的已经是 0x999 了！
