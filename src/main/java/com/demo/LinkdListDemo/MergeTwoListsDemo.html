LeetCode 21: 合并两个有序链表

点击“单步执行”开始。我们将看到 Dummy Node（哨兵节点）的奇妙用法。

ListNode dummy = new ListNode(-1);
ListNode curr = dummy;
// 当两个队伍都有人时，比较队首
while (list1 != null && list2 != null) {
if (list1.val <= list2.val) {
curr.next = list1;
list1 = list1.next;
} else {
curr.next = list2;
list2 = list2.next;
}
// curr 往前走一步
curr = curr.next;
}
// 把剩下没排完的队伍直接接在后面
curr.next = list1 != null ? list1 : list2;
return dummy.next;
